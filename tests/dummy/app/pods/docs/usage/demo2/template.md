# Demo 2

Demo 2 uses Phaser3's physics engine to demonstrate gravity and object collisions. 

For this playable game, use the following keys to move and collect the stars.
 
- 'A' to move left
- 'D' to move right
- SPACE or 'W' to jump.  
 
 Avoid the bombs after level 1!
